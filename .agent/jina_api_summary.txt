**1. What is Jina Embeddings API?**
* **Purpose:** Provides top-performing multimodal multilingual long-context embeddings for search, RAG (Retrieval-Augmented Generation), and agent applications.
* **Performance:** Outperforms proprietary models from Google, OpenAI, and Voyage AI, especially on visually rich document retrieval.
* **Latest Version:** "v4: Universal Embeddings for Multimodal Multilingual Retrieval" is the most significant leap, a 3.8B model for text and images.

**2. How to Use the Jina Embeddings API:**
* **Authentication:** Requires an API key. You can get a free trial key or purchase a paid plan.
    * **Note:** The example `curl` command includes `Authorization: Bearer " "` suggesting it needs an API key.
* **Input:**
    * A single API call can process up to **2048 sentences or texts**.
    * Max input length for a single sentence: **8192 tokens**.
    * **Multimodal Input (for `jina-clip` models):** You can send images using either `url` (URL of the image) or `bytes` (base64 encoded image) in the `input` field.
* **Output:**
    * Embeddings can be floats (default), or you can choose `binary` for faster retrieval or `base64` for faster transmission.
    * **L2 normalization** can be applied (enabled by default) to scale embeddings to a Euclidean (L2) norm of 1, useful for dot-product operations, classification, and visualization.

**5. Pricing and Token Usage:**
* The pricing model is based on the **total number of tokens processed**.
* **Text tokens:** Standard token counting.
* **Image tokens:** Calculated based on "tile size" and "coverage" of the image.
    * `jina-embeddings-v4`: 28x28 pixel tiles, 10 tokens per tile.
    * `jina-clip-v2`: 512x512 pixel tiles, 4000 tokens per tile.
    * `jina-clip-v1`: 224x224 pixel tiles, 1000 tokens per tile.
* **Rate Limits:** Vary by product and API key type (w/o API Key, w/ API Key, w/ Premium API Key) and are measured in RPM (requests per minute) and TPM (tokens per minute).
* **Free Trial:** Available with 10 million tokens.
* **No charge for failed requests.**

**6. Integrations:**
The Embedding API is "natively integrated with various renowned databases, vector stores, RAG, and LLMOps frameworks." Examples of integrated tools include:
* **Vector Stores:** MongoDB, DataStax, Qdrant, Chroma, Weaviate, Milvus, Epsilla, MyScale, LanceDB, TiDB.
* **RAG:** LlamaIndex, Haystack, Langchain, Dify.
* **Observability:** Portkey, Carbon.

**7. On-premises Deployment:**
Jina Embeddings models can be deployed on AWS SageMaker, Microsoft Azure, and soon Google Cloud Services. Commercial licenses are available for private cloud and on-premises servers.

**8. Key Advantages Highlighted:**
* **Long Context:** 8192-token input length.
* **Multilingual and Multimodal:** Covers both text and images across many languages.
* **Performance:** High-performing models that lead benchmarks.
* **Compatibility:** API endpoint matches OpenAI's `text-embedding-3-large` for easy migration.
